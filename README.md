# Love Page 💖

Trang web kỷ niệm tình yêu được làm riêng cho:
- **Đặng Ngọc Tiến**

## Nội dung chính
- Hero section lãng mạn
- Bộ đếm thời gian yêu nhau theo thời gian thực
- Lá thư tình cá nhân hóa
- Timeline kỷ niệm
- Album ảnh và video
- Hiệu ứng trái tim / ánh sáng / nền lung linh
- Pop-up xem ảnh lớn (lightbox)
